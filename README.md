Мобильное приложение для учета семейных финансов
Это мобильное приложение, разработанное с использованием React Native, предназначено для учета семейных финансов. Приложение предоставляет пользователю удобный интерфейс для управления доходами и расходами, анализа финансовых данных и визуализации финансовых трендов с помощью диаграмм.

Функции
1 Учет доходов: добавление и хранение записей о доходах с указанием категории и суммы.
2 Учет расходов: добавление и хранение записей о расходах, отображение их в списке и диаграммах.
3 Финансовый анализ: визуализация доходов и расходов с помощью круговых и столбчатых диаграмм.
4 Безопасное хранение данных: все данные надежно сохраняются локально с помощью Expo SecureStore.
5 Сохранение данных: данные сохраняются между сессиями приложения и восстанавливаются при перезапуске.
6 Очистка данных: возможность удалить все сохраненные данные через окно подтверждения.

Установка
1 Клонируйте репозиторий или скачайте исходный код.
2 Убедитесь, что у вас установлен Node.js.
3 Установите зависимости.

Ключевые библиотеки
1 React Native для разработки приложения.
2 Expo SecureStore для безопасного хранения данных.
3 react-native-chart-kit для отображения диаграмм (круговых и столбчатых).
4 React Navigation для управления навигацией в приложении.

Экраны
1 Экран "Доходы": позволяет пользователю добавлять доходы и просматривать их в виде диаграмм.
2 Экран "Расходы": позволяет пользователю добавлять расходы и видеть их в диаграммах.
3 Экран "Анализ": отображает сравнение доходов и расходов через столбчатую диаграмму и показывает баланс.
